ARC Decision
14

ARC — First Time

ARC Decision

What to do:

Review all evaluations and agency decisions. Then choose one option:

If everything is complete: Select "No" (no additional information needed) → Click Send to Board submission
If more information is needed: Select "Yes" → Pick which units should re-review in the table → Click Request additional information

→ Goes to: Board submission or back to the applicant for additional info

Complementary Information (CI) Flow
15

Applicant — Additional Information

CI Flow

What the applicant does:

  1. Fills in the requested additional information
  2. Uploads any required documents
  3. Clicks Validate send page

→ Goes to: Only the selected units re-evaluate (others are skipped)

16

Selective Re-evaluation

CI Flow

What happens:

Only the units selected by ARC will re-evaluate. For example, if only Legal and Business were selected:

  • Legal and Business receive the file for re-evaluation
  • Technical, Compliance, NOC, and agency approvals are automatically skipped

Each selected unit follows the same process: review → write recommendation → approve.

→ Goes to: ARC (second time)

17

ARC — Second Time

CI Flow

What to do:

  1. Review the updated evaluations and new documents
  2. Click Send to Board submission

→ Goes to: Status Letter and remaining steps

Post-ARC Steps
18

Remaining Steps

Post-ARC

After ARC approves, the file goes through:

  1. Status Letter — Generate the official status letter
  2. Signature — Sign the letter
  3. CI Status Letter — Notify the applicant
  4. Board — Board reviews and votes
  5. CEO — CEO signs off
  6. Board (final) — Final board confirmation
  7. SEZ Documents — Issue the final certificate

→ Process complete. The zone is established.

Key Things to Remember

  • 1. Always click the "Processing" tab first. The action buttons are on the Processing tab, not on the Documents tab.
  • 2. For agency approvals (TAJ, JCA, MOFPS): Select "No objection" or "Objection" from the dropdown before clicking the green button.
  • 3. If a button is greyed out: Save any open table rows first (click the checkmark), then try the button again.
  • 4. ARC selective routing: The table lets you pick which units re-review. Only those units will receive the file back — the rest are automatically skipped.
