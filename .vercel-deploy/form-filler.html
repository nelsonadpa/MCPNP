How to Use

Choose the mode that fits your workflow.

  1. Open any eRegistrations service form (Part A) in your browser
  2. Click "eR Form Filler" in your bookmarks bar
  3. A panel appears on the right side of the page
  4. Click "⚡ Auto Fill" for random valid data, OR select a scenario (Demo / Test) and click "Fill Form" for preset data
  5. Click "📎 Fill Files" to upload placeholder PDF documents to all file fields
  6. Review the filled data, then submit the form
Tip: Auto Fill works on any form with zero setup. Preset Fill requires presets to be loaded first (Step 3 above or custom import).
  1. Open a Part B task (Document Review, Evaluation, Approval, etc.)
  2. Click "eR Form Filler" in your bookmarks bar
  3. Click "⚡ Auto-Pilot" — it will:
    • Detect which role you are on
    • Fill all fields with appropriate data
    • Highlight the action button with a green glow
  4. Click the highlighted button to proceed to the next role
  5. Repeat for each role in the workflow
Tip: Auto-Pilot uses role presets when available. For best results, load the Seed Presets first (Step 3 above).
  1. Generate a preset: ask Claude Code to run /fill-form for your service
  2. Copy the JSON output that Claude produces
  3. Open the bookmarklet panel on any eRegistrations page, go to the Manage tab
  4. Paste the JSON into the import area and click Import
  5. Now use Fill Form with your custom preset for that service
Tip: Custom presets persist in your browser's localStorage across sessions. You can export and share them with teammates.

Frequently Asked Questions

Does this modify real data?
No. It only fills form fields in your browser. Nothing is saved to the server until you click Submit.
Does it work on production?
Yes, but use with caution. It is designed for test and staging environments. If you use it on production, review all filled data before submitting.
Can I add my own presets?
Yes. Use the Manage tab in the bookmarklet panel to import custom JSON presets. You can also generate them automatically with the /fill-form command in Claude Code.
What about file uploads?
Click "Fill Files" in the bookmarklet panel to upload a placeholder PDF to every file field. The PDF is generated in-memory and uploaded via the Form.io API.
It didn't fill some select dropdowns
Some dropdowns load their options from the server asynchronously. Try clicking Auto Fill again after a few seconds, or navigate to the tab first and then fill.
Which browsers are supported?
Chrome, Firefox, Edge, and any Chromium-based browser. Safari may work but is not officially tested.
